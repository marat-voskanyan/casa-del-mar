# Casa del Mar — AI Translation Flow Upgrade

Read CLAUDE.md for full project context.
Complete ALL tasks automatically without asking questions.
Run `npm run build` only once at the very end.

---

## The Problem
Current AI generator asks Groq to write in 3 languages at once.
Russian and Armenian quality is poor.

## The Solution
Three-step flow:
- Step 1: Groq writes English (best at English)
- Step 2: DeepL translates English to Russian (world best translator)
- Step 3: Groq translates English to Armenian only (focused single call)
Note: DeepL does NOT support Armenian language.

---

## TASK 1 — Install DeepL SDK

```bash
npm install deepl-node
```

Add to .env.local:
```
DEEPL_API_KEY=your_deepl_key_here
# Get free key from: https://www.deepl.com/pro-api
# Free tier: 500,000 characters per month
```

Add to .env.example the same line.
Add DEEPL_API_KEY to Vercel environment variables after deploying.

## TASK 2 — Rewrite the API Route

Completely rewrite app/api/admin/generate-description/route.ts
with the new three-step flow.

The route should:

1. Validate GROQ_API_KEY and DEEPL_API_KEY exist
2. Parse and validate request body (name and location required)
3. Build features text from selected pills
4. Get style prompt based on selected style

STEP 1 - Generate English with Groq:
- Use model: llama-3.1-8b-instant
- System prompt: "You are a real estate copywriter. Write only the requested description text. No JSON. No labels. No extra text."
- User prompt includes all property details and style instructions
- Ask for ONLY English text output, no JSON, no other languages
- Extract the plain text response

STEP 2 - Translate English to Russian with DeepL:
- Use deepl-node Translator
- translateText(englishText, 'en', 'ru')
- After translation replace "Property highlights:" with "Преимущества квартиры:"

STEP 3 - Translate English to Armenian with Groq:
- Use a SEPARATE Groq call focused only on translation
- System prompt must be very strict:
  "You are an expert Armenian translator.
  You MUST use proper Armenian Unicode characters only.
  Never use Latin letters or transliteration.
  Translate ONLY. Return ONLY the Armenian text.
  If text has Property highlights: translate as Գույքի առավելությունները:
  Keep bullet points with *"
- User prompt: "Translate this to Armenian using only Armenian Unicode: [englishText]"
- After response replace "Property highlights:" with "Գույքի առավելությունները:"

Return JSON:
{
  success: true,
  descriptions: { en, ru, hy },
  steps: {
    english: "Generated by Groq Llama",
    russian: "Translated by DeepL",
    armenian: "Translated by Groq (DeepL does not support Armenian)"
  }
}

Style prompts for each style option:

casa-del-mar style:
"Write in this exact structure:
1. Opening: property type with X bed and X bath in location
2. Overall character and purpose
3. Most unique feature
4. One key lifestyle benefit
5. Section header: Property highlights:
6. Bullet list with * of 4-6 specific facts
Tone: professional and factual. No superlatives. 80-120 words."

luxury style:
"Premium aspirational tone. Focus on lifestyle and exclusivity. 80-100 words."

investment style:
"Professional investment tone. Mention rental yields 6-10 percent, ROI. Use numbers. 80-100 words."

family style:
"Warm practical tone. Focus on family lifestyle, space, safety, pool, beach. 80-100 words."

short style:
"Maximum 2-3 short punchy sentences. Bold and direct. Under 50 words."

detailed style:
"Comprehensive description covering every feature and angle. 120-150 words."
